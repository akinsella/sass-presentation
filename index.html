Faiblesses de CSS3

Faiblesses de CSS3

  • Fichiers sans fin souvent bordéliques (Maintenabilité difficile)
  • Pas de support des variables, mixins, ... (Fonctionnalités manquantes)
  • Les sélecteurs vous dites ? DRY, connais pas ! (Beaucoup de répétitions)
  • ...

Les variables CSS ne seront pas disponibles à l'utilisation tant que IE 7, 8, 9, 10, 11 ne sera pas descendu en dessous de 1% !

Faiblesses de CSS3

La gestion de multiples fichiers CSS est essentielle pour la maintenabilité et la collaboration.
Mais la notation @import rend votre page lente (Multiples requêtes HTTP).

Nous avons besoins d'outils pour les concaténer. Mais peut-être pouvons-nous en attendre mieux ?
C'est ce que vous propose les préprocesseurs CSS !

Les fonctionnalités

A propos des fonctionnalités

Less, Sass et Stylus les supporte toutes.

Ils n'améliorent pas les CSS, mais aident à l'amélioration du workflow de développement.

Les variables

Attention: Les variables sont des constantes 

   $my_blue: #4e6cb0; // Sass
   @my_blue: #4e6cb0; // Less
     my_blue = #4e6cb0; // Stylus

   $primary_color: $my_blue;

   h1 {
    color: $primary_color;
   }

Les types de variables

Les variables ne sont pas que des couleurs 

   $fonts: Helvetica, Arial, sans-serif;
   $image_path: "../img/";
   $font-size: 1.5em;
   $margin: 20px;
   $width: 80%;

Opérations et fonctions de couleur

Maths et opérations sur les couleurs 

   width: 25px * 4 + 100px / 2 - 50px; /* 100px */
   color: #990033 + #666666; /* #FF66CC */
Fonction sur les couleurs 

   $light_blue: lighten($my_blue, 20%);
   $flashy_blue: saturate($my_blue, 50%);

Mixins

Les fonctions telles que lighten() retourne une valeur. Les Mixins ne retournent rien mais génèrent du CSS 

   @mixin border-radius($radius: 10px) {
        -webkit-border-radius: $radius;
           -moz-border-radius: $radius;
                border-radius: $radius;
   }

   .button {
        @include border-radius(4px);
   }

Nesting

Permet de calquer sur la structure des styles sur la structure HTML. Il est préférable de limiter à 4 niveaux maximum. 

    .pop {
        .pop-btn {

        }
    }

    // ======== Output =========

    .pop {}
    .pop .pop-btn {}
                           

     .link {
         &:hover {

         }
     }

     // ======== Output =========

     .link {}
     .link:hover {}

Imports et minification

Les imports fonctionnent sans coût sur les performances 

   @import "colors";
   @import "typography";
   @import "layout";
Le CSS généré peut être clair ou bien minifié 

   #main{color:#fff;background-color:#000}#main
   p{width:10px}.huge{font-size:10em;font-weight:bold;text-decoration:underline}
